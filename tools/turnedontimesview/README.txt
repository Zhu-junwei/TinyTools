TurnedOnTimesView 是一款由 NirSoft 开发的免费工具，用于查看 Windows 计算机的开关机时间历史记录。它可以显示计算机的开机、关机、休眠和恢复时间等详细信息，这对于系统管理员、IT 支持人员或对计算机使用情况感兴趣的用户非常有用。

主要功能：
开机和关机时间：显示计算机的每次开机和关机时间，包括正常关机、重启和强制关机。
休眠和恢复时间：记录计算机进入休眠或睡眠状态以及从这些状态恢复的时间。
事件日志来源：从 Windows 事件日志中提取相关信息，确保数据的准确性。
导出功能：支持将数据导出为文本、CSV、HTML 和 XML 格式，方便进一步分析和保存记录。
便携性：无需安装，下载后即可运行，非常方便。
