Super ACiD Block Attack（SABA）是一款 MS-DOS 俄罗斯方块游戏。
1997 年，本游戏名为 ACiD Tetris，因为版权问题被俄罗斯方块公司勒令移除。
2002 年，本游戏以 Super ACiD Block Attack 的新名称重新发布。
本游戏使用经典俄罗斯方块玩法，在图像和音乐方面都表现良好。


【操作】
默认键位：左右键横移，下键软降，A 逆时针旋转，S 顺时针旋转。
游戏结束后按Shift+Esc退出到菜单选项。

【玩法】
生存、升级、打分。
消行是本游戏的唯一得分途径。
基础得分：消一 100，消二 300，消三 600，消四 1200。
每消 10 行升级速度，每升一级加一倍消行基础分。
消行的实际得分是按消行后的总消行数对应的等级加分的。[注 2]
右上角提示各方块的历史数量。
左下角会有反映游戏表现的互动表情出现。
死亡判定：重叠死亡。

【部分细节说明】
本游戏的长按灵敏度较高，单点时间稍长就会触发重复操作，单旋时需要小心。
本游戏支持同步多元长按。
例如，在开局之前，就把左、软降、逆时针旋转都按住，方块就会连续地快速向左下翻滚。
本游戏的互动表情能够体现出 C2 的计时器连击和背靠背消四的概念，但消行得分不会额外增加。
如果旋转、自动降落和锁定这三个动作的发生时间非常接近，游戏可能会出现砖格错位 bug。
