====================================
 Rebex Tiny FTP Server
====================================

A minimalist FTP server for testing purposes.

https://www.rebex.net/tiny-ftp-server/


## Usage
-------------------------------------
Just run the executable and press the Start button.
Configure it by editing RebexTinyFtpServer.exe.config if needed.
