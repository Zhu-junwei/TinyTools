# TinyTools - 小而美的工具集

**TinyTools** 集多个小巧实用的工具于一身，装机必备的工具集。基于 [CLaunch](https://ss1.xrea.com/pyonkichi.g1.xrea.com/en/)
，在[th_sjy](http://www.th-sjy.com/?p=1973)汉化基础上收集的各种小工具。

## 特性

- **小而美**：  
  TinyTools 平均每个工具不超过 2MB。无需安装，解压后即可运行，轻巧便捷。  
  还内置了 11 款皮肤，让你使用时更加美观。

- **可插拔**：  
  所有工具都位于 `tools` 目录下，工具之间相互独立（DOSBox 除外）。如果你只需要某个工具，可以直接将其复制到自己的电脑中，
  删除或添加自己的软件，打造一个符合自己需求的工具集。非常适合用作系统的快速导航工具。

TinyTools 目前集成了以下工具分类：

## 下载与安装

1. 下载 **TinyTools** 压缩包。
2. 解压缩到任意目录。
3. 首次运行点击`首次使用先点我.bat`进行初始化配置并会默认开启工具，你可以在系统的任务栏找到显示的图标。
4. 下次使用可以直接双击`CLaunch.exe`或`ClAdmin.exe`以管理员身份运行。
5. 默认提供三种唤醒方式，可以根据自己的喜好选择或在选项中进行更改：
    - Ctrl + Shift + q
    - 桌面左键双击空白处
    - 单击鼠标中间
